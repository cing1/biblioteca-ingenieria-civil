# 📚 Biblioteca de Ingeniería Civil - USFX

> Plataforma web para la recopilación y distribución de libros, solucionarios y apuntes de libre acceso para estudiantes y profesionales de Ingeniería Civil.

## 📖 Descripción
Este proyecto funciona como una biblioteca digital centralizada que facilita el acceso rápido a material académico de Ingeniería Civil. Los recursos están alojados en Google Drive y se cargan dinámicamente en la interfaz web, permitiendo una navegación intuitiva y sin costos adicionales para la comunidad universitaria.

## ✨ Características
- 📂 **Recursos organizados:** Libros, solucionarios y apuntes disponibles por categorías.
- ☁️ **Integración con Google Drive:** Carga automática y sincronizada desde almacenamiento en la nube.
- 🆓 **Acceso libre y gratuito:** Material 100% abierto para fines educativos.
- 📱 **Interfaz ligera:** Diseño web sencillo, rápido y compatible con múltiples dispositivos.

## 🚀 Cómo usar
1. Accede a la URL donde está alojado el proyecto.
2. Explora las categorías disponibles (ej. `Todo…`).
3. Selecciona el libro, solucionario o apunte que necesites.
4. Haz clic para descargar o visualizar el archivo directamente desde Google Drive.

## 🛠️ Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript (Vanilla)
- Google Drive API / Enlaces compartidos
- Hosting estático
